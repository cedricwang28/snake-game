Rules of the Game

1. Click to open each block，if there is a bomb，game failed.

2. If there is a number on the block，it represents how many mines there are in the surrounding eight blocks.

3. To mark a mine that you think there is in a block，press shift+click.

4. If mines around a number have all been marked，double click the number to open the remaining blocks.

5. When your marks match all mines, you win.

6. Click face to restart.
